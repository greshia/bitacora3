Frameworks

Un frameworks es un conjunto de herramientas, librerías, convenciones y buenas prácticas que pretenden encapsular las tareas repetitivas en módulos genéricos fácilmente reutilizables. Se utilizan en el ámbito de programación de aplicaciones desde hace décadas.

Un framework CSS es un conjunto de herramientas, hojas de estilos y buenas prácticas que permiten al diseñador web olvidarse de las tareas repetitivas para centraste en los elementos unicos de cada diseño en los que puede aportar valor.

SVG

Es un lenguaje de marcado para describir imágenes y aplicaciones de gráficos bidimensionales, y un conjunto de interfaces de scripts de gráficos relacionados. Es un formato de gráficos libre de derecho de uso generalizado desarrollado y mantenido por el W3C working group. Se pueden utilizar con imágenes como formas, trazados, texto y efectos de filtro.

Mobile first

Estar conectado en internet se volvió parte de nuestro día a día. Poco a poco estamos dejando de usar la computadora para realizar las tareas cotidianas y estamos usando cada vez más los smartphones. Los dispositivos móviles garantizan la practicidad que buscamos, pero, para que la experiencia en internet sea buena, es necesario que la navegación sea pensada para estos dispositivos. Lo más sensato seria crear primero un sitio web optimizado para los dispositivos móviles. Hace más sencilla la tarea de diseñar cualquier página web, de manera gradual el diseño se ira acomodando a los distintos tamaños de pantalla, adaptando el layout al tiempo. Las pantallas grandes son menos practico.

Modulo grid

El módulo Grid de CSS es una herramienta muy poderosa y flexible para diseñar y organizar el contenido de una página web en una rejilla (grid). Permite crear diseños de múltiples columnas y filas de manera sencilla y eficiente, sin tener que recurrir a flotantes o posicionamiento complejo. Con CSS Grid, puedes dividir un contenedor en filas y columnas de manera explícita y controlar cómo los elementos hijos se distribuyen dentro de esa rejilla.

Unidades CSS

Unidades absolutas, como px (pixeles), cm (centímetros) e in (pulgadas), representa tamaños fijos y no cambian independientemente del contexto en el que se utilicen, proporcionando una medida exacta.
Unidades relativas. Las unidades relativas, como % (porcentaje), em y rem, son flexibles y se adaptan en función de otros elementos. Por ejemplo, % es relativo al tamaño del contenedor padre, em es relativo al tamaño de fuente del elemento padre, y rem es relativo al tamaño de fuente del elemento raíz.

Supports

Sirve para detectar la compatibilidad de características CSS en diferentes navegadores y apliquen estilos condicionales basados en esta compatibilidad para asegurar una experiencia de usuario constante.
La regla supports en css permite detectar si el navegador soportar una característica css especifica y aplicar estilos condicionales basados en esta compatibilidad.
